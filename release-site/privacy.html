Privacy

Your files stay
on your Mac.

Headroom is designed to explain storage without turning your file list into somebody else’s data.

What Headroom reads

Headroom checks file locations, names, modification dates, logical sizes, allocated sizes, and cloud-download status in the folders you allow it to monitor. Duplicate detection reads file contents locally to calculate exact-match hashes.

What Headroom stores

Scan summaries, recommendation preferences, protected paths, cleanup history, and optional feedback are stored in your local Application Support folder.

What leaves your Mac

Nothing is uploaded by the app. Diagnostic exports are created only when you request one. File paths are excluded unless you explicitly enable them, and you choose whether to share the exported file.

What Headroom can change

By default, Headroom only observes. If you enable approved cleanups, it acts only on the files shown in the review. Regular cleanup files move to Trash; Headroom never empties Trash. Supported iCloud actions remove only the local download, not the cloud copy.

Alpha software

Headroom is currently an alpha. Review every suggested file and keep current backups of important data. Report privacy or safety concerns through the project’s issue tracker.
